# 遇到的坑

参考了阮一峰的博客教程，遇到了一些坑记录一下

* `USERNAME` 是包括 `@163.com` 的
* `PASSWORD` 是 163 的 **授权密码**，不是 **登陆密码** [ref](https://blog.csdn.net/archer119/article/details/64942220)
* 仓库中和教程的 `secrets` 命名不同，差了一个下划线，评论中已有人指出。
